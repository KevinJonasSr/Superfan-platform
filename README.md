# Superfan Platform

A fan engagement and monetization platform for artists.

## Overview

This platform enables artists to:
- onboard fans
- reward engagement
- sell exclusive experiences
- build community
- track fan activity

## Core Features

### Fan Experience
- onboarding wizard
- fan dashboard
- rewards & tiers
- marketplace
- referrals
- community hub

### Artist/Admin Experience
- admin dashboard
- rewards manager
- marketplace manager
- fan CRM
- analytics

## Tech Stack (Planned)

- Next.js
- PostgreSQL
- Prisma
- Supabase Auth
- Stripe
- Tailwind
