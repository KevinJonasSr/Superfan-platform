# MVP Build Specification

## Goal
Build the first working version of the Superfan Platform that allows:
- fans to join and engage
- fans to earn rewards
- fans to purchase offers
- artists to manage the experience

## Core Features

### Fan Side
- onboarding wizard
- fan dashboard
- rewards & tiers
- marketplace
- referrals

### Admin Side
- admin dashboard
- rewards manager
- marketplace manager
- basic analytics
